# Changelog

## [0.1.0] - 2025-12-02

### Added
- Multiple weather providers (NWS for US, Open-Meteo for international)
- Automatic provider selection based on location
- Temperature unit conversions (Celsius, Fahrenheit, Kelvin)
- Weather condition mapping from WMO codes and NWS text
- Swift 6 concurrency support with actor-based design
- Cross-platform support (iOS, macOS, tvOS, watchOS, visionOS)
- Retry mechanism with exponential backoff via swift-retry
- City name geocoding support
- Real-time weather update streams
